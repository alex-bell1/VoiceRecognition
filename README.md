# 🔊 Python Text-to-Speech App

A simple and customizable text-to-speech application built with Python and `pyttsx3`. The app converts user-entered text into speech, allows adjustment of speech rate and volume, and lets users choose from available system voices.

---

## ✨ Features

* 🗣️ Convert text into speech instantly
* 🎚️ Adjustable speech rate
* 🔊 Adjustable volume control
* 🎤 Multiple voice selection support
* ⚡ Lightweight and beginner-friendly
* 🛡️ Input validation and error handling

---

## 🛠️ Requirements

* Python 3.x
* pyttsx3

Install dependencies:

```bash
pip install pyttsx3
```

---

## 🚀 Usage

Run the application:

```bash
python text_to_speech.py
```

The program will:

1. Display all available system voices
2. Ask for text input
3. Ask for speech settings
4. Read the text aloud using the selected voice
